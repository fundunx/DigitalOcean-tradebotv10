# ApexQuant V10

ApexQuant V10 is a safe paper-trading crypto trading platform.

## Safety

- Paper trading only.
- Live trading disabled.
- Capital protection first.
- Cash is allowed.
- No forced trades.
- Every trade must have a reason, confidence, stop loss, target, expected value, and risk check.

## Commands

npm test
npm run paper:soak
npm start

## API

GET /health
GET /api/state
GET /api/readiness
GET /api/settings/risk
